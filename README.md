# Skyblock Fusion Profits

A live Hypixel Skyblock attribute fusion calculator using the Bazaar API.

## Features

* Live Bazaar price integration
* Fusion profit/hr calculations
* Hunter Fortune support
* Attribute modifier support
* Liquidity bottleneck calculations
* Real-time ranked fusion results
